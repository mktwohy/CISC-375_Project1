Synth

Android app for additive synthesis. After adjusting the harmonic-series editor and waveshape selector to construct a sound, the user can play music with the resizable, polyphonic keyboard and pitch bend slider.

The app uses a synth library I wrote in a Kotlin for generating, manipulating, and playing periodic signals with overtones.

Tone Tutor is an Android app that utilizes pitch detection and Fourier analysis to quantify timbre. It aims to beginner musicians with real-time, visual, and numerical feedback to train their ear and improve their technique.

Satisfactory Calculator

Satisfactory is a factory building game that I've been playing for the past few months. The game requires some simple ratio math that starts to become tedious when planning larger factories. So, using Compose for Desktop, a Kotlin framework, I created a desktop app for browsing the game's recipes and figure out clock speeds for a given output/minute or vice versa. As a bonus, the app's UI mimics an in-game menu.

Note - this project is unfinished; changing the production rate does not update the other stats as it should.
